# Employee Management Dashboard

## Project Overview
This project is a React-based Employee Management Dashboard.  
It allows users to manage employee records with authentication, CRUD operations, filtering, and print functionality.

## Features
- Mock login authentication
- Protected dashboard access
- Add, edit, and delete employees
- Profile image upload with preview
- Active / Inactive employee status
- Search employees by name
- Filter employees by gender and status
- Combined search and filter functionality
- Print employee list
- Data persistence using LocalStorage

## Tech Stack
- React.js
- Vite
- JavaScript
- React Router
- Context API
- LocalStorage (used as mock API)

## Data Handling
Employee data is stored in LocalStorage to simulate backend API behavior.  
All CRUD operations persist across page reloads.
